floor-0x72.png and wall-0x72.png are single tiles cropped from the full sprite
sheet (0x72_DungeonTilesetII_v1_3.png, one point release behind the current
v1.7). Since itch.io gates the download behind a $0 "pay what you want"
checkout that isn't a plain file link, the sheet was instead fetched from a
public GitHub mirror that vendors the same CC0 asset as game art:
https://github.com/yujqiao/DungeonRush/blob/master/res/drawable/0x72_DungeonTilesetII_v1_3.png
(that repository's own code is GPLv3, but the CC0 terms above are what govern
this specific asset, independent of the license of whatever project embeds it).
